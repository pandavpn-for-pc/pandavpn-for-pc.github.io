Initial setup & configuration (after install)

Login / Subscription

PandaVPN supports freemium and subscription models. If you have a paid subscription, sign in with the same account used on mobile. For Play Store purchases, subscription will be managed by Google Play if you use the Android APK inside an emulator and tie the emulator to your Google account.

Choosing protocols & settings

  • Use WireGuard or UDP protocols for speed (if available).
  • Enable AES-grade encryption or ECC where options exist — security vs performance tradeoffs.
  • Enable Kill Switch to prevent traffic leakage if the VPN connection drops.
  • Use split tunneling for apps that should not route through VPN (e.g., local banking apps if they block VPNs).

Streaming & geo-unblocking

For streaming, pick servers optimized for the target streaming service (US servers for US Netflix, UK servers for BBC iPlayer). Streaming performance can fluctuate depending on server load — test several servers and protocols if you face buffering.

Gaming & low-latency mode

Choose servers close to the game server region and prefer UDP/WireGuard protocols that minimize handshake overhead and latency. Close background bandwidth-heavy apps and ensure emulator is allocated enough CPU/RAM.

Troubleshooting & Tips

  • Connection drops: try a different server, switch protocol, or temporarily disable kill-switch to diagnose.
  • Slow speeds: test servers closer geographically; enable WireGuard; check ISP throttling.
  • Emulator networking issues: use "bridged" network mode if the emulator supports it or ensure host firewall allows emulator processes.
  • Streaming not working: choose a server labeled for streaming or clear cookies / use a fresh browser profile inside the emulator.

If you still face issues, consult PandaVPN support (email listed on Play Store) or use official help channels.
